# Qt项目功能概述设计文档——第35组

1. 组长：周诣超
2. 队名：原子爆弹（dàn）
3. 项目名称：原子爆弹（tán）
4. 项目创意：一款融合了弹球和合成两大元素的小游戏。玩家通过鼠标操作初始的氢原子，碰撞屏幕上的其他原子，有概率产生核聚变合成原子序数较大的原子，合成出铁原子（比结合能最大）即获得游戏胜利。如果碰撞上原子爆弹（dàn），原子就会爆炸，分解成原子序数小的原子或消失，碰到墙或其他原子则反弹。不同的原子具有不同的特性。同时我们计划加入多个模式与成就系统，保留双人联机的可能。
5. 主体功能：见上所述。
6. 包含模块：教程，经典模式，限时模式，双人模式等。
7. 大致时间节点和人员分工：
   1. 游戏具体机制设计——梁昱桐，周诣超，李弘杰
   2. 碰撞公式推理和实现——梁昱桐
   3. 成就系统——周诣超，李弘杰
   4. 技能设计——李弘杰
   5. 界面及动画设计——周诣超，李弘杰
   6. 音效设计——梁昱桐
   7. 教程制作——周诣超
